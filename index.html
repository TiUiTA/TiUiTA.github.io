Enterprise Ready:

  • Run thousands of tests in parallel on hundreds real devices without human interaction
  • Distribute tests across devices to speed up test executions
  • Support hundreds of users writing and running tests. No centralized server going down and leaving your team blocked
  • Manage your tests on source control as any other software artifact
  • Generate useful documentation from Features and Scenarios to share across stakeholders
  • Continuous Integration support. Easy to integrate
  • Rich and beautiful test result reports
  • Web based Live monitoring for parallel execution of multiple devices

Write test at the level of abstraction you want...

  • ... from Feature files (Gherkin specification style)
  • ... or RSpec BDD style
  • ... to low level WebDriver API scripts

We recommend Feature and Gherkins style tests, based on easy to use CoffeeScript DSL

  • No need for expert developers, any QA, even Product Owners can write tests easily
  • Use existing, open source, and feature rich IDEs or simple text editors, your choice

Requirements and Compatibility:

  • Android 2.x and greater (Devices, Emulators and Genymotion)
  • iOS 3.x and greater (Devices and Simulators)
  • No need to root or jailbreak your devices
  • No complex setup needed
  • No servers required
